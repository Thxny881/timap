  Subfinder
  Waybackurls
  Uro

  How to Install:
  
    1.Click in the file timap.sh
    2.Click in the Download Raw File.
    3.Open the Terminal.
    4.Use the command:  chmod 777 timap.sh && sudo cp timap.sh /usr/bin

    Command: timap.sh
